Important Question

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a way that web browsers represent a webpage so that programming languages, like JavaScript, can interact with it.

How Do you select an Element from the DOM?

You can select elements from the DOM using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. Each method helps you find the parts of a webpage you want to work with, making it easy to change or interact with them!

What is event Delegation in context of Dom, Why it is useful?

Event delegation is a technique in JavaScript that allows you to handle events for multiple elements using a single event listener. Instead of attaching individual event listeners to each element, you attach one listener to a parent element, and it captures events that bubble up from its child elements.


Event delegation is useful because it improves performance by reducing the number of event listeners, allowing for better memory usage and faster response times.

How Do you manipulate an element's Attributes and Styles using DOM?

To manipulate an element's attributes and styles using the DOM, you can use JavaScript methods. For attributes, you can use setAttribute() to change an attribute's value or getAttribute() to retrieve it. To modify styles, you can access the style property of an element and set CSS properties directly . This allows for dynamic updates to both attributes and styles on the webpage.
